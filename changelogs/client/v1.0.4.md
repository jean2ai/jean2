
### Changed
- **Refactored HTTP data fetching from manual `useEffect`/`useCallback` patterns to TanStack React Query** — added `@tanstack/react-query@5` with a centralized `QueryClient`, query key factory, and query/mutation hooks for all config panels (Tools, Models, Preconfigs, Prompts, Providers, MCP), file browsing (FileTree, FolderPickerDialog, file preview), session lists, and workspace CRUD. WebSocket event handlers now invalidate relevant query caches to keep them in sync with real-time state. Config mutations sync updated data back to the Zustand `serverDataStore` so components reading from either source stay consistent.
