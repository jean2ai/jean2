### Changed

- **Structured output visual overhaul** — structured responses (from response formats and tool visualizations) now have much better visual hierarchy:
  - **Array items as cards** — arrays of objects render as distinct bordered cards with a subtle background instead of flat bullet lists. Each item is visually self-contained, making collections like review feedback or analysis results easy to scan.
  - **Numbered index badges** — each card gets a numbered badge (#1, #2, ...) so individual items are easy to reference in conversation.
  - **Definition-list layout for nested objects** — key-value pairs inside cards use a compact horizontal layout (muted label on the left, value on the right) instead of stacking label above value. Significantly reduces vertical height.
  - **Markdown rendering for string values** — strings containing markdown syntax (bold, code, links, lists, headings, tables, blockquotes) are now rendered as formatted markdown instead of literal text. Plain strings like file paths stay lightweight.
