### Added

- **Gmail provider in OAuth panel** — Gmail now appears as a connectable provider card in Settings > OAuth, alongside Codex. No client code changes were needed — the existing `OAuthProvidersPanel` auto-renders any server-registered connectable provider. Users see the Connect button, manual-paste fallback, and disconnect flow identical to Codex.
- **Archive all sessions in a tag** — each tag group header in the sidebar now has a dropdown menu (three dots, visible on hover) with an "Archive all" action. A confirmation dialog shows the tag name and exact session count before archiving all sessions sharing that tag in one click. Reuses the existing per-session close flow.
- **Tag grouping in Overview** — the Overview view (multi-workspace sidebar) now replicates the full tag structure from the single-workspace view. Sessions within each favorited workspace are grouped by their primary tag in collapsible sections, sorted by most recently updated. Each tag group has a count badge and "Archive all" action. Per-session tag add/remove/change actions are available via the session dropdown menu. Tag collapse state is persisted across views via shared localStorage.
