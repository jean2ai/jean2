### Changed

- **Inline tools selector in preconfig editor** — replaced the Popover + Command (cmdk) tools picker with an inline searchable toggle list. The Popover-inside-Dialog combination caused scroll locking issues that prevented mouse/trackpad scrolling of the tool list. The new inline list renders directly in the form with a search input, scrollable bordered list, and checkbox indicators for each tool. Also removed the redundant selected-tools pills since selection state is now shown inline.
