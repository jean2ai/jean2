### Added
- Revert button is now available on the first user message, allowing you to clear the entire conversation and start fresh.
- "Clear Conversation" confirmation dialog shown when reverting from the first message, with distinct messaging from the normal revert dialog.

### Fixed
- Terminal panel is now accessible from any view within a workspace (workspace landing page, overview, etc.), not just when a session is active. Since the terminal is workspace-bound, it no longer requires an active session to be opened.
