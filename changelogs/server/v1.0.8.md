### Added
- **Response format CRUD API** — new REST endpoints `GET/POST/PUT/DELETE /api/response-formats` with full create, read, update, delete support. Formats are stored in SQLite with JSON Schema definitions. Built-in formats no longer have delete protection — all formats are freely editable and deletable.
- **Structured output in messages** — when a response format is selected, the LLM's structured output is parsed from the stream result and attached to the final assistant message as `structuredOutput` (with `formatName`, `data`, `schema`). Stored in the `structured_output` column on the messages table.

### Changed
- **Removed built-in format protection** — the DELETE endpoint no longer rejects `builtin-` prefixed IDs. All formats are treated equally.
