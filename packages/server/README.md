# Jean2 Server

The AI Agent backend server built with Hono and Bun. Supports multiple LLM providers (Anthropic, OpenAI, Google, OpenRouter, etc.) and Model Context Protocol (MCP).

## Installation

Download from GitHub releases or build from source:

```bash
# Build from source
bun run build

# Run in development
bun run dev
```

## Usage

Start the server:

```bash
bun run start
```

The server runs on port 3000 by default. Configure via environment variables.
