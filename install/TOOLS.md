# Tools

Install tools into `~/.jean2/tools/`. Copy and run the commands below.

## [apply-patch](../tools/apply-patch-bun/README.md) v0.2.1
Apply unified diff patches to files atomically.

```bash
mkdir -p ~/.jean2/tools/apply-patch && curl -L https://github.com/rabbyte-tech/jean2/releases/download/tool-apply-patch-bun%2Fv0.2.1/apply-patch-bun.tar.gz | tar -xz -C ~/.jean2/tools/apply-patch --strip-components=1
```
